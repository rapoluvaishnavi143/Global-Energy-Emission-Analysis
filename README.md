# 🌍 Global Energy Emissions Analysis (SQL)

## 📌 Project Overview

This project focuses on analyzing global energy emissions data using SQL to uncover key trends, identify major sources of pollution, and track changes over time.

The goal is to derive insights that can support environmental awareness and data-driven decision-making.

---

## 🛠️ Tools & Technologies

* MySQL
* SQL (JOINs, CTEs, Window Functions)
* Data Analysis

---

## 🗄️ Database Design

* Designed a structured relational database to store emissions data
* Organized data into tables for efficient querying
* Ensured data consistency and integrity

---

## 📊 Analysis Performed

* Identified top energy sources contributing to emissions
* Analyzed emissions by country and region
* Calculated year-over-year emission trends
* Performed per capita emission analysis
* Used window functions for ranking and trend analysis

---

## 🔍 Key Insights

* CO₂ emissions are the largest contributor globally
* Fossil fuels like coal and oil dominate emission sources
* Emissions show a steady increase over the years
* Significant variation exists between countries

---

## 🚀 Conclusion

This project demonstrates strong SQL skills in querying, data analysis, and extracting meaningful insights from structured datasets.
